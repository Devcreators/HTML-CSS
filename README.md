### 1. HTML 기본 코드

```
  <!DOCTYPE html>
    <html>
      <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width">
        <title>JS Bin</title>
      </head>
      <body>
        <h1>h1</h1>
        <h2>h2</h2>
        <h3>h3</h3>
        <button>button!</button>
      </body>
    </html>
```
